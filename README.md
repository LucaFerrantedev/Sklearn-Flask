Aplicación web desarrollada con Flask que permite la carga y el análisis automatizado de datos socioeconómicos. El sistema procesa un archivo CSV dado en el repositorio para generar un modelo de regresión exponencial que predice la expectativa de vida basándose en índices de ingresos. Incluye un sistema de autenticación de usuarios y visualización dinámica de resultados mediante gráficos generados en tiempo real con Matplotlib.
